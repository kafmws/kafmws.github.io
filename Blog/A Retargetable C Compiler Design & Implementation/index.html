# README

本系列文章是经典书籍《A Retargetable C Compiler: Design And Implication 》 (opens new window)（《可变目标C编译器：设计与实现》）一书的读书笔记。摘录了书中几乎所有代码，加以解释，并记录自己的一些思考。 笔记中的部分代码与与原书中有出入，因为原书代码为lcc 3.x version并且去除了一些assert语句，而笔者所搜集的lcc代码版本为4.2。来自 GitHub (opens new window).
